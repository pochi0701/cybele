# Cybeleダウンロード
Cybeleは、アプリケーションサーバーです。フロントとサーバー側共にJavaScriptで稼働します。
ここではパッケージ化したCybeleをダウンロードできます。

対象OS  
Windows:Windows10,Windows11  
Linux:32bit,64bitのすべてのlinux,Raspberry-pi等も含みます。
